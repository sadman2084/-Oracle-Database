question:Display a list of all instructors, showing each instructors ID and the number of sections taught. Make sure to show the number of sections as 0 for instructors who have not taught any section.

        
            SELECT instructor.ID, 
            COALESCE(COUNT(teaches.sec_id), 0) AS number_of_sections
     FROM instructor
     LEFT JOIN teaches ON instructor.ID = teaches.ID
     GROUP BY instructor.ID
     ORDER BY number_of_sections ASC, instructor.ID ASC;
